# RSRP-Cellular-Networks
RSRP Cellular Network Signal Strength Prediction
Machine learning and deep learning models for predicting Reference Signal Received Power (RSRP) from a private telecom dataset, built during an internship at the University of Glasgow.
Results

98% reduction in RMSE (73.74 → 1.31) through iterative feature selection, regularization, and hyperparameter tuning
94.2% classification accuracy on a 4-class DNN signal strength classifier

Models Built

Linear Regression (baseline and revised)
Dense Neural Network (DNN) classifier
Evaluated activation functions (ReLU vs Leaky ReLU) and scalers (Standard vs Robust)

Tech Stack
Python, TensorFlow/Keras, scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
